## 2. 의도된 사용
### 포함
- 임상 의사결정 **보조** (권고/근거 제공, 최종 판단은 인간)
- 재활 루틴 생성·수정·안전성 점검
- 코드 리뷰/PR 제안(테스트/리스크/롤백 플랜 포함)

### 제외
- 독자적 최종 의료판단 및 처방/시술 결정
- 고위험 실시간 제어(예: 생명유지장치 직접 제어)

## 3. 데이터·학습 개요(요약)

## 4. 평가(회귀 12태스크 + PoU 지표)
